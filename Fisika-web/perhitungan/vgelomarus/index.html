Visualisasi Gelombang Arus AC

Simulasikan dan visualisasikan gelombang arus AC dengan memasukkan parameter-parameter dasar

Parameter Input

Masukkan nilai arus maksimum, frekuensi, dan fase awal untuk memvisualisasikan bentuk gelombang arus

Hasil Perhitungan

Frekuensi Sudut (ω): -- rad/s
Fase Awal (φ): -- rad

Visualisasi Gelombang Arus AC

I(t) = Imax × sin(ωt - φ)
Keterangan Rumus:
  • I(t) = arus pada waktu t (Ampere)
  • Imax = nilai maksimum amplitudo arus (Ampere)
  • ω = frekuensi sudut = 2πf (rad/s)
  • φ = pergeseran fase awal (radian)
